# SWOT Application
Vor dem Ausführen

Beispielhafte Anleitung im Program Visual Studio Code (könnte je nach system variieren):

Öffnen/ Auswählen des Projektordners

Im Terminal
"python -m pip install -r requirements.txt"

Im Gitbash
Zum Ausführen "python -m streamlit run src/app.py"
Zum Programieren wurde ChatGPT genutzt

Link im Gitbash Anklicken um die Local gehostete Webseite zu öffnen.

Loggin:
Der Username ist: admin
Das Password ist: admin

Für den Acess API Key braucht man einen Key von der Google Cloud. Unter anderem erhältlich unter: https://console.cloud.google.com/apis/credentials?project=gen-lang-client-0039768934

Dannach alle Felder ausfüllen und auf "Generate SWOT" drücken. Gegebenenfalls mehr Fragen ausfüllen und auf "continue" drücken.
Alle anderen Funktionen sind durch Linke Mausklicken zugänglich
